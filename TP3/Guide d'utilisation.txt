Guide d'utilisation de l'application (TP3)

Prérequis :
- Avoir installé Node.js
- Avoir installé un navigateur web

Utilisation :
1. Lancer le service de base de données
	1a. Naviguer vers le répertoire "DatabaseService" avec une invite de commande.
	1b. Installer les dépendances avec la commande "npm install".
	1c. Lancer le service avec la commande "npm start".
2. Lancer le service de recherche de musique
	2a. Naviguer vers le répertoire "MusicSearchService" avec une invite de commande.
	2b. Installer les dépendances avec la commande "npm install".
	2c. Lancer le service avec la commande "npm start".
3. Lancer le serveur web
	3a. Naviguer vers le répertoire "MusicBrowserClient" avec une invite de commande.
	3b. Installer les dépendances avec la commande "npm install".
	3c. Lancer le service avec la commande "npm start".
4. Ouvrir un navigateur et entrer l'adresse "http://localhost:8888".
6. Modifier le contenu de la barre de recherche pour lancer une nouvelle recherche. La recherche est lancée automatiquement.

Création de playlist :
1. Cliquer sur l'onglet "Playlist".
2. Entrer un nom de playlist.
3. Cliquer sur le bouton "Create playlist". La nouvelle playlist devrait apparaître dans la section "My playlists".
4. Cliquer sur l'onglet "Musique".
5. Choisir un morceau et cliquer sur "+".
6. Cliquer sur le nom de la nouvelle playlist. Le morceau est maintenant dans la nouvelle playlist.
